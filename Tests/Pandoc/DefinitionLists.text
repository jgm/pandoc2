# Definition Lists

Tight using spaces:

apple
:   red fruit
orange
:   orange fruit
banana
:   yellow fruit

Tight using tabs:

apple
:	red fruit
orange
:	orange fruit
banana
:	yellow fruit

Loose:

apple
:   red fruit

orange
:   orange fruit

banana
:   yellow fruit

Multiple blocks with italics:

*apple*
:   red fruit

    contains seeds,
    crisp, pleasant to taste

*orange*
:   orange fruit

        { orange code block }

    > orange block quote

Multiple definitions, tight:

apple
:   red fruit
:   computer
orange
:   orange fruit
:   bank

Multiple definitions, loose:

apple
:   red fruit

:   computer

orange
:   orange fruit

:   bank

Blank line after term, indented marker, alternate markers:

apple

  ~ red fruit

  ~ computer

orange

  ~ orange fruit

    1. sublist
    2. sublist
